PJMTR – Backend

Backend API for PJMTR, a research and multidisciplinary journal platform that manages paper submissions, application tracking, and admin review workflows.

🚀 Features

Research paper submission (PDF upload)

Auto-generated Application ID

Submission status tracking

Admin panel APIs for review & updates

Secure data storage

JWT-based authentication

🛠 Tech Stack

Node.js

Express.js

MongoDB + Mongoose

JWT Authentication

Multer (PDF uploads)

 Setup
git clone https://github.com/your-username/pjmtr-backend.git
cd pjmtr-backend
npm install
npm start


Server runs on:

http://localhost:5000
